这里在 Fedora Linux 上搭建，其他操作系统的步骤也是差不多的。

首先使用内置的包管理器查看是否是最新版本（dnf 是 Fedora 内置的包管理器），是则通过dnf install hugo命令安装，否则到 GitHub 下载最新版本 Hugo（hugo_extended_withdeploy_x.xxx.x_linux-amd64.tar.gz）
