Fedora 安装 fcitx5

Created2025-04-05|Updated2025-04-05
|Post Views:

参考 fcitx5，RIME，rime-ice

安装 fcitx5

使用系统内置的包管理器安装（dnf 是 Fedora 内置的包管理器）：

雾凇拼音

雾凇拼音——长期维护的简体词库

小鹤双拼（可选）

我使用小鹤双拼方案，所以这里修改一下，用全拼的可以跳过这一步

修改候选词数量

默认候选词数量是 5，有需要可以修改

关闭候选词拼音注释

默认候选词会有拼音注释，比如：你好[ni hao]，编辑 double_pinyin_flypy.schema.yaml（编辑哪个文件看你使用什么方案），注释下面这两行就可以关闭

然后到 Fcitx5 Configuration 添加 Rime，调整字体大小为 16 或者更大，通过 GNOME Tweaks 设置开机启动，自带的英文键盘可以删除

配置环境变量（可选）

Fedora Workspace 41 没添加也能正常使用，如果需要可以添加到 ~/.bashrc 或者 ~/.profile

重新加载

重启整个 fcitx5 框架，如果没起作用可以重启或者重新登录
