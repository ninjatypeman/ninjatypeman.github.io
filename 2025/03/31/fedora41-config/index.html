Fedora 快捷键

  • 切换到第一个/最后一个 workspace：WIN + Home/End
  • 左上角滑动、Dock 上滑动
  • 切换 workspace：WIN + PgUp/PgDn
  • 先按一下 WIN，松开，使用 PgUp/PgDn 切换
  • Alt + F6 切换一个应用的多个窗口
    • Alt + Esc 在当前 workspace 切换窗口（可以实现 Alt + F6 的效果）
