Hexo 博客搭建（2025）

Created2025-03-30|Updated2025-04-02
|Post Views:

Hexo 是一个快速、简洁的静态博客框架，搭建起来相对简单。下面是使用 Hexo 搭建博客的基本步骤：

  1. 安装 Hexo
  1. 初始化 Hexo 博客
  1. 安装依赖
  1. 启动本地服务器
  1. 生成 SSH 密钥对，然后把生成的 .pub 文件的内容添加到 GitHub
  1. 创建 GitHub 仓库（Public），仓库名：<username>.github.io
  1. 安装部署插件
  1. 修改根目录下的_comfig.yml配置文件
  1. 生成静态文件、部署
  1. 新建博客文章
  1. 新建草稿
  1. 创建另一个 Private 仓库保存源码，以便在其他设备上编辑、发布（注意.gitignore 文件）
